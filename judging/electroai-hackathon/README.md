# ElectroAI Hackathon Judging Materials

This folder contains judging rubric materials adapted from the temporary Clinical AI Hackathon workspace for reuse in the ElectroAI Hackathon.

## Structure

- `source/judging-criteria.md`: the written judging criteria and scoring sheet source.
- `source/print-scoring-rubric-prompt.md`: the prompt used to generate the printable scoring rubric.
- `output/hackathon-scoring-rubric.html`: the print-ready HTML scoring rubric.
- `output/hackathon-scoring-rubric.pdf`: the PDF export of the scoring rubric.
- `output/hackathon-clinician-scoring-sheets.pdf`: the clinician-facing scoring sheets.

## Notes

- No separate executable generation script was present in the source workspace.
- The rubric generation workflow in the source project was prompt-based, using `source/print-scoring-rubric-prompt.md` together with `source/judging-criteria.md`.
